# Moonhook V4 :lucide-moon-star: 

Welcome to the official documentation for **Moonhook V4**. This documentation covers the installation, usage and plugin development of **Moonhook V4**.

Moonhook features:

<div markdown>
  - :material-language-cpp: __Source code fully written in C++__
  <br>
  - :lucide-rocket: __Fast automation and execution__
  <br>
  - :material-language-lua: __Fast *Luau* plugin system__
  <br>
</div>

**Moonhook V4** is a full rewrite of **[Moonhook V3](https://github.com/Seal-Softworks/MoonhookV3)** in :material-language-cpp:.

!!! warning "Moonhook V3"
    **Moonhook V3** is no longer supported or updated. Please, use **Moonhook V4** instead.

### Installation

=== "Windows"
    To install **Moonhook V4** for Windows, take a look at the [Installation Manual](installation/windows.md).
=== "Other"
    To install **Moonhook V4** for any other OS or architechture, you will need to compile Moonhook from the source code yourself. Take a look at the [Installation Manual](installation/other.md). To get instructions on how to compile and run **Moonhook V4**.
